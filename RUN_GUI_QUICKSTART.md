# RUN_GUI_QUICKSTART

## ตั้งค่า virtual environment
1. ตรวจสอบว่าเครื่องติดตั้ง Python 3.10+ ที่มี tkinter (ติดมากับตัวติดตั้ง Windows)
2. เปิด PowerShell ในโฟลเดอร์โปรเจกต์
3. รัน python -m venv .venv
4. รัน .venv\\Scripts\\Activate.ps1

## ติดตั้งแพ็กเกจที่ต้องใช้
- รัน pip install -r requirements.txt

## รันกราฟิกอินเทอร์เฟซ
- รัน python gui_app.py
- หากต้องการปิด virtual environment ให้ใช้คำสั่ง deactivate
